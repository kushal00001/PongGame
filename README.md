# PongGame (Unity)

## Requirements
To run or build this project, you need:
- **Unity Hub**
- **Unity Editor version 2021.3 LTS or later**
- **Windows Build Support module** (for creating the executable)
- A computer capable of running Unity and basic games

---

## How to Download the Project
1. Go to the GitHub repository:
   https://github.com/kushal00001/PongGame
2. Click the green **Code** button
3. Click **Download ZIP**
4. Extract the ZIP file to a folder on your computer

---

## How to Open the Project in Unity
1. Open **Unity Hub**
2. Click **Add**
3. Browse to the extracted folder and select the project root folder (the folder containing `Assets/`)
4. Click **Open**

---

## How to Build the Game (Create executable)
1. In Unity, go to:
   **File â†’ Build Settings**
2. Select **PC, Mac & Linux Standalone**
3. Set **Platform** to **Windows**
4. Click **Build and Run**
5. Choose a folder name (example: `Build`)
6. Unity will create:
   - `pong.exe`
   - `pong_Data/`

---

## How to Run the Game
If you built the game:
1. Open the folder containing `pong.exe`
2. Double-click **pong.exe**

If running in Unity:
1. Click the **Play** button in the Unity editor

---

## How to Play
- Use the **Up Arrow** and **Down Arrow** keys to move the paddle
- The ball will bounce off the walls and paddle
- If the ball goes past the paddle, the game will reset automatically

---

## Help Screen (Pause Menu)
- Press **H** at any time during the game
- The help screen will appear and the game will pause
- Press **H** again to close the help screen and resume the game

---

## How to Quit the Game
Press the **Esc** key at any time to quit the game.

---

## Notes
- If the paddle is not visible in the built executable, adjust the screen resolution to **1280 x 720** in the build settings.
- Make sure the build file `pong.exe` is placed in the project root before submission.
